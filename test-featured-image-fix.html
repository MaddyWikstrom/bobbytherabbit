Problem: Your site was using the first image from Shopify's images array as thumbnails, but Shopify has a designated "featured image" for each product that should be used as the thumbnail.
Solution: Updated the GraphQL queries and image processing logic to prioritize Shopify's featured image as the thumbnail.

üîß Files Modified

  • netlify/functions/get-products.js - Added featuredImage field to GraphQL query
  • scripts/products.js - Updated image processing to prioritize featured image

üìù GraphQL Query Changes

Added the following field to fetch Shopify's designated featured image:

featuredImage { url altText }

üîÑ Image Processing Logic

The new logic ensures that:

  • If a product has a featured image set in Shopify, it becomes the first image (thumbnail)
  • If the featured image is already in the images array, it gets moved to the front
  • If the featured image is not in the images array, it gets added at the beginning
  • This ensures the thumbnail always matches what's set in Shopify admin
